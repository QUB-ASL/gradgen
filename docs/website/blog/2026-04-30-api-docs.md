The API docs of gradgen are now available [here](https://qub-asl.github.io/gradgen/api-dox/). The API docs have been generated automatically from the docstrings in the code using [Sphinx](https://www.sphinx-doc.org/en/master/).

<!-- truncate -->

For example, you can find detailed API docs for the [`SingleShootingBundle`](https://qub-asl.github.io/gradgen/api-dox/api/gradgen.html#module-gradgen.single_shooting) class, [`SquaredDistanceToSet`](https://qub-asl.github.io/gradgen/api-dox/api/gradgen.html#gradgen.squared_distance_to_set.SquaredDistanceToSet) and a lot more. 

The API docs are still under development. Currently, the majority of public API endpoints (classes, methods, functions) have API docs. The next step will be to include examples and code snippets everywhere.

Note that best way to get started with gradgen is our [online documentation](/gradgen/docs/category/learn), not the API docs.
