# Squared distance to set


Let $C$ be a nonempty, closed, convex set. The squared (Euclidean) distance-to-set function, 
```math
\operatorname{dist}_{C}^{2}(x) = \inf_{y \in C} \|y - x\|^{2},
```
is well defined.
Its gradient is 

```math 
\nabla \left(\tfrac{1}{2}\operatorname{dist}_{C}^{2}(x)\right) = x-\Pi_C(x), 
```

where $\Pi_C$ is the projection onto $C$.

Squared distance-to-set functions often play an important role in 
optimization problems. Here we will see how to define such functions. 
Gradgen also offers a easy constructors for common sets $C$ such as 
norm-balls.
