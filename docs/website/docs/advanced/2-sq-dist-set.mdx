# Squared distance to set


Let $C$ be a nonempty, closed, convex set. The squared (Euclidean) distance-to-set function, 
```math
\operatorname{dist}_{C}^{2}(x) = \inf_{y \in C} \|y - x\|^{2},
```
is well defined.
Its gradient is 

```math 
\nabla \left(\tfrac{1}{2}\operatorname{dist}_{C}^{2}(x)\right) = x-\Pi_C(x), 
```

where $\Pi_C$ is the projection onto $C$.

Squared distance-to-set functions often play an important role in 
optimization problems. Here we will see how to define such functions. 
Gradgen also offers easy constructors for common sets $C$ such as
Euclidean balls, infinity-norm balls, and axis-aligned rectangles.

## Custom sets

The user needs to specify the squared distance-to-set function 
and the projection operator. For simple sets, such as norm-balls,
rectangles, etc, see the section on [common convex sets](/gradgen/docs/advanced/sq-dist-set#common-convex-sets) below.

As an example, suppose $C = \{x\in\mathbb{R}^2: x_2 = 0\}$.
Then,
```math
\operatorname{dist}_{C}^{2}(x) = x_2^2,
```
and 
```math
\Pi_{C}^{2}(x) = (x_1, 0).
```
Firsly, we need to define the above functions as instances of [`Function`](/gradgen/docs/basics/functions)

```python
# Symbol
x = SXVector.sym("x", 2)

# Projection function
projection = Function(
    "projection",
    [x],
    [SXVector((x[0], SX.const(0.0)))],
    input_names=["x"],
    output_names=["p"],
)

# Squared distance-to-set function
sq_distance = Function(
    "sq_distance",
    [x],
    [0.5 * x[1] * x[1]],
    input_names=["x"],
    output_names=["d"],
)
```

Then, we need to bundle the above functions in a `SquaredDistanceToSet` 
as follows

```python
distance = (
    SquaredDistanceToSet(name="dist_to_axis")
    .with_projection_function(projection)
    .with_sq_distance_function(sq_distance)
)
```

This object can be used as a function. For example, we can do

```python
distance_value = distance([1., 100.])
```

We can also define the gradient of the distance function 
and call it as a function 

```python
grad_dist = distance.gradient()
grad_dist_val = grad_dist([1., 1.00])
```

We can also use the `distance` in other functions (we can compose 
it with other functions). For example, we can define the function 
$g(x) = \mathrm{dist}_C^2(x / \|x\|^2)$ and compute its derivative:

```python
g = Function(
    "magic",
    [x],
    [distance(x / x.norm2sq())],
    input_names=["x"],
    output_names=["g"],
)
dg = g.gradient()
dg_val = dg([1., 1.00])
```


## Common convex sets

### Norm balls

Gradgen provides factory constructors for the most common convex sets.
For example, a Euclidean ball centered at the origin with radius 1 can be
written as

```python
distance = SquaredDistanceToSet.euclidean_ball(
    name="unit_ball",
    center=(0.0, 0.0),
    radius=1.0,
    input_name="z",
)
```

An infinity-norm ball uses the same calling pattern:

```python
distance = SquaredDistanceToSet.infinity_ball(
    name="unit_box",
    center=(0.0, 0.0),
    radius=1.0,
)
```

For axis-aligned rectangles, the bounds may include ``-inf`` on the lower
side and ``+inf`` on the upper side:

```python
distance = SquaredDistanceToSet.rectangle(
    name="half_infinite_box",
    xmin=(-1.0, float("-inf")),
    xmax=(float("inf"), 2.0),
)
```
