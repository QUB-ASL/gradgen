# Installation

## Install gradgen

It is recommended that you use a virtual environment with Python 3.13.
Install gradgen using

```bash
pip install gradgen
```

## Install Rust

To use the auto-generated Rust code, you need to have installed 
Rust on your system. You will find instructions [here](https://rust-lang.org/tools/install/).
After you install Rust, we recommend that you also install `cargo-fmt` with 

```bash
rustup component add rustfmt
```
