# SquaredDistanceToSet Demo

This demo shows how to define a projection-backed half-squared distance with
`SquaredDistanceToSet` and use it inside an ordinary symbolic `Function`.

## Set and distance

The represented set is the x-axis in $\mathbb{R}^2$:

$$C = \{(x_1, x_2) \in \mathbb{R}^2 : x_2 = 0\}.$$

Its projection map is

$$\Pi_C(x_1, x_2) = (x_1, 0).$$

The demo uses the half-squared distance

$$\tfrac{1}{2}\mathrm{dist}_C(x)^2 = \tfrac{1}{2} x_2^2,$$

whose gradient is

$$\nabla \tfrac{1}{2}\mathrm{dist}_C(x)^2 = x - \Pi_C(x).$$

## Composition

To show that `SquaredDistanceToSet` composes like any other primitive, the
demo defines

$$f(x) = \tfrac{1}{2}\mathrm{dist}_C(2x)^2.$$

The generated Rust crate exposes:

- the primal kernel for `f(x)`
- the gradient with respect to `x`

## Files

- [`main.py`](./main.py): defines the distance primitive, evaluates it in
  Python, and generates the Rust crate
- [`squared_distance_to_set_kernel/`](./squared_distance_to_set_kernel/):
  generated by running `main.py`
- [`runner/`](./runner/): a small Rust binary crate that depends on the
  generated kernel crate and calls the primal and gradient kernels

## Running the demo

From the repository root, after activating your virtual environment:

```bash
PYTHONPATH=src python demos/squared_distance_to_set/main.py
```

This generates:

```text
demos/squared_distance_to_set/squared_distance_to_set_kernel
```

To run the Rust example:

```bash
cd demos/squared_distance_to_set/runner
cargo run
```
