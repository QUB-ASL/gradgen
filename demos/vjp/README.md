# Jacobian And VJP Demo

This demo shows two related reverse-mode ideas for a vector-valued function
$G : \mathbb{R}^2 \to \mathbb{R}^3$.

We define

$$G(x) =
\begin{bmatrix}
x_1 + x_2 \\
x_1 x_2 \\
\sin(x_2)
\end{bmatrix},
\qquad
x =
\begin{bmatrix}
x_1 \\
x_2
\end{bmatrix}.$$

Its Jacobian matrix is

$$J_G(x) =
\begin{bmatrix}
1 & 1 \\
x_2 & x_1 \\
0 & \cos(x_2)
\end{bmatrix}.$$

For a cotangent vector

$$v =
\begin{bmatrix}
v_1 \\
v_2 \\
v_3
\end{bmatrix},$$

the vector-Jacobian product is

$$J_G(x)^\top v =
\begin{bmatrix}
v_1 + x_2 v_2 \\
v_1 + x_1 v_2 + \cos(x_2) v_3
\end{bmatrix}.$$

The demo shows how to:

1. evaluate the Jacobian in Python as a flat row-major vector
2. build a runtime-seeded `Function.vjp(wrt_index=0)`
3. generate one Rust crate containing the primal function, the Jacobian, and
   the VJP kernel

## Files

- [`main.py`](./main.py): builds the symbolic function, evaluates the flat
  Jacobian and VJP, and generates Rust code
- `vjp_kernel/`: generated by running `main.py`
- `runner/`: a small Rust binary crate that depends on `vjp_kernel` by path,
  calls the primal, Jacobian, and VJP kernels, and prints their outputs

## Running the demo

It is advisable to run the demo from an activated virtual environment.

From the repository root, after activating your virtual environment:

```bash
python demos/vjp/main.py
```

This will:

- print one flat row-major Jacobian and one VJP evaluation
- generate a Rust crate in:

```text
demos/vjp/vjp_kernel
```

To see how the generated crate can be used from Rust in practice, run:

```bash
cd demos/vjp/runner
cargo run
```
