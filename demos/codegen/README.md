# Rust Code Generation Demo

This demo shows the basic gradgen workflow:

1. define a symbolic function in Python
2. evaluate it in Python
3. generate a Rust crate that evaluates the same function

The example function is

$$f(x, u) = \Vert x \Vert_2^2 + u_1 \sin(x_1) + x_2 x_3,$$

where:

- $x = (x_1, x_2, x_3)$ is a 3D input vector
- $u = (u_1)$ is a 1D input vector

This is a good minimal example because it includes:

- a quadratic term, $\Vert x \Vert_2^2 = x_1^2 + x_2^2 + x_3^2$
- a nonlinear term, $u_1 \sin(x_1)$
- a simple cross term, $x_2 x_3$

## Files

- [`main.py`](./main.py): defines the function and generates Rust code
- `codegen_kernel/`: generated by running `main.py`

## Running the demo

It is advisable to run the demo from an activated virtual environment.

From the repository root, after activating your virtual environment:

```bash
python demos/codegen/main.py
```

This will:

- print the function value for one sample input
- generate a Rust crate in:

```text
demos/codegen/codegen_kernel
```
