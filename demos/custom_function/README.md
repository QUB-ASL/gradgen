# Opaque Custom Function Demo

This demo shows how to register a custom scalar-valued function
$f:\mathbb{R}^2 \times \mathbb{R}^2 \to \mathbb{R}$, where:

- $x = (x_1, x_2)$ is the function argument
- $w = (w_1, w_2)$ is a parameter vector

The example is intentionally non-trivial and uses an opaque Python/NumPy
implementation for the derivatives.

## Function definition

We define
$$
f(x, w) = 2^{w_1} x_1^2 + w_2 x_2^2 + \sin(x_1 x_2).
$$

This combines:

- a parameterized quadratic term in $x_1$
- a weighted quadratic term in $x_2$
- a nonlinear coupling term $\sin(x_1 x_2)$

## Jacobian

Since $f$ is scalar-valued, its Jacobian with respect to $x$ is the row
vector of first derivatives:
$$
J_x f(x, w) =
\begin{bmatrix}
2 \cdot 2^{w_1} x_1 + x_2 \cos(x_1 x_2) \\
2 w_2 x_2 + x_1 \cos(x_1 x_2)
\end{bmatrix}.
$$

## Hessian

The Hessian with respect to $x$ is
$$
H_x f(x, w) =
\begin{bmatrix}
2 \cdot 2^{w_1} - x_2^2 \sin(x_1 x_2) &
\cos(x_1 x_2) - x_1 x_2 \sin(x_1 x_2) \\
\cos(x_1 x_2) - x_1 x_2 \sin(x_1 x_2) &
2 w_2 - x_1^2 \sin(x_1 x_2)
\end{bmatrix}.
$$

## Hessian-vector products

For a direction $v \in \mathbb{R}^2$, the Hessian-vector product is $H_x f(x, w)\, v.$

In the demo code, this is implemented as an opaque Python callback using a
matrix-vector multiplication with the Hessian above.

## Files

- [`main.py`](./main.py): registers the custom function, evaluates it in
  Python, and generates a Rust crate
- [`custom_function_kernel/`](./custom_function_kernel/): generated by running `main.py`

## Running the demo

It is advisable to run the demo from an activated virtual environment.

From the repository root, after activating your virtual environment:

```bash
PYTHONPATH=src python demos/custom_function/main.py
```

This will print sample evaluations and generate a Rust crate in:

```text
demos/custom_function/custom_function_kernel
```
