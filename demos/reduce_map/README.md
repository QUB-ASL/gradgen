# Map + Reduce Demo

This demo introduces `reduce_function(...)` and shows how it combines naturally
with `map_function(...)` in a loop-preserving staged pipeline.

The demo constructs three layers:

1. A scalar map kernel `m(x)`
2. A scalar reduce stage kernel `r(a, m)`
3. A composed map-then-reduce pipeline

The generated Rust code preserves loops (`for stage_index in 0..N`) rather than
expanding into unrolled stage-specific algebra.

## Mathematical setup

Let `count = N` and let the packed scalar input sequence be

$$x_{\mathrm{seq}} = (x^{(1)}, x^{(2)}, \dots, x^{(N)}).$$

Define the map stage

$$m(x) = \sin(x) + x^2.$$

Applying map stage-wise yields

$$m_{\mathrm{seq}} = \bigl(m(x^{(1)}), m(x^{(2)}), \dots, m(x^{(N)})\bigr).$$

Define the reduce stage transition

$$r(a, u) = \sin(a) + a\,u + u^2.$$

Given an initial accumulator value $a^{(0)} = a_0$, reduce computes a left fold

$$a^{(k+1)} = r\bigl(a^{(k)}, m(x^{(k+1)})\bigr), \quad k=0,\dots,N-1.$$

The final output is

$$a_{\mathrm{final}} = a^{(N)}.$$

Equivalently, with explicit fold notation,

$$a_{\mathrm{final}} = \operatorname{reduce}_{r,N}\!\left(a_0,\operatorname{map}_{m,N}(x_{\mathrm{seq}})\right).$$

## What `reduce_function` means in gradgen

For a binary stage function

- input 1: accumulator shape `A`
- input 2: stage input shape `X`
- output: next-accumulator shape `A`

`reduce_function(stage_fn, N, ...)` constructs a staged source that accepts:

- an initial accumulator `acc0` with shape `A`
- a packed sequence `x_seq` of length `N * |X|`

and returns the final accumulator with shape `A`.

The output shape must match the accumulator input shape.

## Demo files

- [`main.py`](./main.py): defines map/reduce/pipeline functions and generates Rust
- `reduce_map_kernel/`: generated by running `main.py`
- `runner/`: Rust binary consuming the generated crate by path

## Running from Python

From the repository root, after activating your virtual environment:

```bash
python demos/reduce_map/main.py --count 4
```

This will:

- evaluate mapped, reduced, and composed map+reduce functions in Python
- evaluate gradients of the composed pipeline in Python
- generate a Rust crate in `demos/reduce_map/reduce_map_kernel`

## Running the Rust runner

```bash
cd demos/reduce_map/runner
cargo run
```

The runner prints kernel metadata and sample primal/gradient outputs from the
`reduce_map_kernel` crate.
