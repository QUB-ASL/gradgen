# Python Interface Demo

This demo shows how to generate a Rust crate that can be imported from Python.

The generated crate:

- exposes a PyO3 module named `foo`
- provides `workspace_for_function("energy")`
- provides `call("energy", x, w, workspace)`
- returns a dictionary when the generated function has multiple outputs

## Files

- [`main.py`](./main.py): generates the Python-enabled Rust crate
- `foo/`: generated by running `main.py`
- [`runner/main.py`](./runner/main.py): creates an isolated Python virtual
  environment, installs the generated crate, and calls the exported module

## Running the demo

From the repository root, after activating your virtual environment:

```bash
python demos/python_interface/main.py
python demos/python_interface/runner/main.py
```

The first command generates the crate in `demos/python_interface/foo`.
The second command installs that crate into an isolated temporary virtual
environment and prints the result of calling `foo.call(...)` from Python.
